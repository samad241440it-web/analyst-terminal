# Analyst Terminal Dashboard

## Overview
Analysts deal with continuously evolving data streams. Detecting unusual patterns quickly is difficult.

## Our Solution
We built a real-time analyst dashboard that visualizes evolving data streams and highlights unusual signals.

## Features
- Live updating market chart
- Animated market ticker
- Signal detection system
- Activity log tracking events
- Clean terminal-style UI

## Tech Stack
HTML  
CSS  
JavaScript  
Chart.js

## How to Run
Open index.html using Live Server in VS Code.
